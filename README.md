# Jêbas Lanches - Menu Online 🍔

Este é o sistema de cardápio online do **Jêbas Lanches**, projetado para facilitar o processo de pedidos via WhatsApp com um visual premium e funcionalidade de recibo para produção.

## 🚀 Funcionalidades

- **Cardápio Interativo**: Seleção simples de itens com descrições e preços.
- **Carrinho Dinâmico**: Cálculo automático de subtotais e total do pedido.
- **Checkout Integrado**: Campo para endereço de entrega e observações.
- **Integração com WhatsApp**: Formatação automática do pedido no estilo de "ticket de produção" (impressora não fiscal).
- **Design Responsivo**: Experiência otimizada para Desktop e Mobile.

## 🛠️ Tecnologias Utilizadas

- **HTML5** para estrutura semântica.
- **CSS3** para estilização premium (Glassmorphism e Layout Responsivo).
- **JavaScript (Vanilla)** para lógica de pedidos e integração.
- **Fontes Google** (Inter e Oswald) e **FontAwesome** para ícones.

## 📦 Como Usar

1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador.
3. Comece a escolher os lanches e finalize via WhatsApp!
